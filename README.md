# go-codeact-agent

A CLI-based AI coding agent built in Go that performs automated code reviews using the [CodeAct](https://arxiv.org/abs/2402.01030) pattern. It connects to an LLM (via Groq) and uses a tool-calling loop to read files, inspect git diffs, and execute PowerShell scripts — all autonomously within your local repository.

## How It Works

1. You provide a prompt describing what you want reviewed.
2. The agent sends it to the LLM along with available tools (git diff, file reading).
3. The LLM calls tools or outputs PowerShell scripts to gather context.
4. Scripts are executed locally and results are fed back into the conversation.
5. The loop continues until the LLM outputs `[DONE]`, delivering a structured Markdown review.

## Prerequisites

- [Go 1.25+](https://go.dev/dl/)
- [Git](https://git-scm.com/)
- PowerShell (comes pre-installed on Windows)
- A [Groq API key](https://console.groq.com/keys)

## Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/lbalbas/go-codeact-agent.git
   cd go-codeact-agent
   ```

2. **Create your `.env` file**
   ```bash
   cp .env.example .env
   ```
   Then edit `.env` and add your API keys:
   ```env
   GROQ_API_KEY="your-groq-api-key-here"
   ```

3. **Install dependencies**
   ```bash
   go mod download
   ```

## Usage

Run the agent from the root of any git repository you want to review:

```bash
go run ./cmd/agent
```

You'll be prompted to enter a task description. The agent will then autonomously gather context and produce a code review.

## Adding New Tools

To add a new tool the LLM can call:

1. Create a handler function that accepts a JSON arguments string and returns a string result.
2. Register the tool's schema in the tool configuration.
3. Map the tool name to its handler in the tool registry.

## Security Note

> **⚠️ This agent executes PowerShell scripts generated by an LLM on your local machine.** Use it in controlled environments and review the agent's output carefully. Never run it with elevated privileges or on production systems.
